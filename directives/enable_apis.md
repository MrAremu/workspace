# Enable Google APIs

Your project requires the following APIs to be enabled. Please visit the links below and click **ENABLE** for each one.

1.  **People API** (For contact lookup)
    *   [Enable People API](https://console.developers.google.com/apis/api/people.googleapis.com/overview?project=146166226382)

2.  **Google Drive API** (For copying/exporting files)
    *   [Enable Drive API](https://console.developers.google.com/apis/api/drive.googleapis.com/overview?project=146166226382)

3.  **Google Docs API** (For editing invoices)
    *   [Enable Docs API](https://console.developers.google.com/apis/api/docs.googleapis.com/overview?project=146166226382)

4.  **Google Sheets API** (For logging)
    *   [Enable Sheets API](https://console.developers.google.com/apis/api/sheets.googleapis.com/overview?project=146166226382)

5.  **Gmail API** (For sending emails)
    *   [Enable Gmail API](https://console.developers.google.com/apis/api/gmail.googleapis.com/overview?project=146166226382)

After enabling these, run the test script again.
